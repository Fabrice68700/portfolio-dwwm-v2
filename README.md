# Portfolio Moderne - Fabrice Ansel (V2)

Ce projet est une refonte complète de mon portfolio d'apprentissage, conçu pour être à la fois un outil de présentation professionnelle et un support pédagogique. 

## ✨ Nouveautés de la Version 2
- **Design Premium** : Utilisation du **Glassmorphism** (effets de verre dépoli) et de dégradés modernes.
- **Sémantique & Accessibilité** : Structure HTML5 renforcée avec attributs ARIA pour une meilleure inclusion.
- **Interactivité Dynamique** : Animations d'entrée au défilement via l'API **Intersection Observer**.
- **Typographie Fluide** : Adaptation automatique de la taille du texte sans media queries complexes (via `clamp()`).
- **Pédagogie Intégrée** : Chaque ligne de code est commentée pour expliquer les concepts de développement web.

## 🛠️ Technologies & Concepts
- **Frontend** : HTML5 (Sémantique), CSS3 (Flexbox, Grid, Variables), JavaScript (ES6+).
- **Concepts clés** : Responsive Design, Mode Sombre (LocalStorage), Animations CSS, Manipulation du DOM.

## 🚀 Objectif AFPA
Ce portfolio documente mon parcours jusqu'à mon entrée en formation **Développeur Web et Web Mobile (DWWM)** à l'AFPA de Mulhouse en octobre 2026.
